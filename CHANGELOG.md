# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-12-18

### Added
- Core MCP server implementation with 5 tools.
- Curated game templates for Fantasy, Sci-Fi, Mystery, and Horror-Lite.
- Deterministic seed-based RNG for shareable runs.
- survival mechanics (HP, Supplies, Inventory).
- React-based UI widgets for MCP clients.
- GitHub Actions CI workflow.
- Cyberpunk and Surreal game templates.
- Project hygiene files (LICENSE, CONTRIBUTING, SECURITY).

### Changed
- Refactored `TemplateManager` to load templates from JSON files instead of being hardcoded.
- Improved `README.md` with architecture diagrams and detailed tool documentation.
